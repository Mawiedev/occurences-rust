# occurences-rust 

Count the number of occurrence of the given word in the string and print the number of occurence in the word.

Example:

aaabbc --> a3b2c
aaabaaa --> a3ba3
abcde --> abcde
abcdeee --> abcde3
abbbbbcdeee  --> ab5cde3
a -> a
aaaaaa -> a6 
"" --> ""

## Test it 

For testing with other words, you have to replace the string in the "result()" function.
